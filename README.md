# TouchNet: Glove-Controlled Phone Service

TouchNet is a innovative phone service that allows you to control your mobile device using a wearable glove. By leveraging touch-sensitive points on the glove, you can perform various actions on your phone without directly interacting with the device. This documentation provides an overview of the project, installation instructions, and usage guidelines.

## Inspiration

The idea behind TouchNet was born out of the need for enhanced accessibility and control of mobile devices in situations where direct phone access is difficult or impractical. Whether it's for personal safety, physical limitations, or convenience, TouchNet aims to provide an alternative control mechanism that enhances the user experience.

## Key Features

- **Wearable Glove:** The specially designed glove incorporates touch-sensitive points that detect user input accurately.
- **Bluetooth Connectivity:** Establishes a reliable and secure connection between the glove and the paired phone.
- **Background Service:** A dedicated mobile application runs in the background, listening for incoming Bluetooth data from the glove.
- **Action Mapping:** Users can customize the touch points on the glove and map them to specific actions on their phones.
- **Versatile Use Cases:** TouchNet can be utilized in emergency situations, for individuals with physical disabilities, or to improve convenience and efficiency.

## Installation

To use TouchNet, follow these steps:

1. Clone the TouchNet repository to your local machine.
2. Install the mobile application on your phone from the provided APK file.
3. Pair your phone with the wearable glove via Bluetooth.
4. Launch the TouchNet mobile application and grant necessary permissions.

## Usage

1. Wear the glove and ensure it is connected to your phone via Bluetooth.
2. Launch the TouchNet mobile application, which will run in the background.
3. Customize the actions associated with each touch point on the glove using the intuitive interface.
4. Perform touch gestures on the glove to trigger the predefined actions on your phone.
5. Enjoy the convenience and control offered by TouchNet in various situations.

For more detailed instructions and tips, refer to the [User Manual](/docs/user-manual.md).
